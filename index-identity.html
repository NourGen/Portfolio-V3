Portfolio Visual Identity

This page defines the visual identity of your portfolio: primary and secondary colors, typography choices, and how each token should be used for consistency.

Color System

Primary Color

#C4783A

Use for main CTA buttons, active links, highlights, and focus states.

Secondary Color

#D4914E

Use for secondary highlights, gradient blends, and decorative accents.

Accent Color

#E8B88A

Use sparingly for hover states, pale highlights, and soft decorative details.

Background Base

#1A0E08 / #FAF7F2

Dark: warm brown-black. Light: warm ivory. Both anchor the BSA identity.

Surface/Card

#2A1810 / #EDE3D6

Use for cards, containers, and panels above the base background.

Typography System

Cormorant Garamond — Display Font (Headings)

DM Sans — Body Font (paragraphs, labels, and navigation). This pairing keeps headings scholarly and refined while preserving readability in long text.

  • Heading font: Cormorant Garamond (weights 600–700)
  • Body font: DM Sans (weights 300–500)
  • Text color: #FAF7F2 on dark / #1A0E08 on light
  • Muted text: #9C7B65 on dark / #7A5C45 on light

Brand Rule (Quick)

  • Copper (#C4783A) dominates interactive elements — buttons, links, active states.
  • Warm gold (#D4914E) supports visuals but should not overpower primary actions.
  • Keep generous spacing and warm surfaces to preserve the premium BSA look.
  • Never use cold greys or blues — all neutrals stay within the warm brown family.
